# Kaboom

**Kaboom** is a Swift-based Xcode project created as a learning and experimentation playground for iOS/macOS development. 
> The project focuses on exploring application structure, Swift language features, and Apple’s testing frameworks, with both unit tests and UI tests included.


---

##  Features

- Built with **Swift**
- Xcode project structure
- Unit tests included
- UI tests included
- Clean, extensible project layout

---

##  Project Structure

```text
.
├── Kaboom.xcodeproj          # Xcode project file
├── Kaboom/                  # Main application source code
├── KaboomTests/             # Unit tests
├── KaboomUITests/           # UI tests
└── README.md                # Project documentation
```
## Prerequisites  
- macOS
- Xcode (latest stable version recommended)
- Swift 5+
